# Halloween
## Origens
A origem do *Halloween* é uma mistura de tradições celtas e cristãs, originadas principalmente nas Ilhas Britânicas. 
Os celtas celebravam o festival pagão Samhain em **31 de outubro**, marcando o fim do verão e o início do inverno, um período em que acreditavam que o **véu entre o mundo dos vivos e dos mortos se tornava fino**. Posteriormente, a Igreja Católica moveu o Dia de Todos os Santos para 1º de novembro e a véspera desse dia, 31 de outubro, se tornou a **"All Hallows'** Eve", ou Halloween.

## Tradições Modernas
Hoje, o *Halloween* é comemorado de maneira divertida e criativa em diversos países. As pessoas se fantasiam, decoram suas casas e participam de festas temáticas. Entre as tradições mais populares estão:

- **Doces ou Travessuras** (Trick or Treat): crianças vão de porta em porta pedindo doces.  
- **Fantasias Criativas**: adultos e crianças vestem-se como monstros, heróis ou personagens famosos.  
- **Abóboras Iluminadas**: conhecidas como Jack-o’-lanterns, são esculpidas e decoradas com rostos assustadores.

## CSS Grid
*CSS Grid* é uma especificação do W3C que permite criar layouts de página web bidimensionais (*em linhas e colunas*) de forma mais eficiente e flexível. **Ele funciona definindo um "contêiner" de grid e, em seguida, usando propriedades para organizar os "itens" (*elementos filhos*) dentro das células da grade, permitindo posicioná-los e dimensioná-los livremente sem depender de técnicas antigas como floats.** 

## CSS Flexbox
*Flexbox* (*Flexible Box Layout*) é um módulo de layout **CSS** para criar layouts *unidimensionais*, como linhas ou colunas, de forma eficiente para interfaces de usuário. Ele permite que os itens dentro de um contêiner flexível sejam **organizados, alinhados e distribuídos de forma responsiva**, podendo crescer ou diminuir para preencher o espaço disponível. 

## Jogos
Existem jogos como **Flexbox Froggy** e **Grid Garden** para praticar o aprendizado de *CSS Grid e Flexbox*, respectivamente. Esses jogos ensinam os conceitos de forma interativa, com desafios que exigem o uso das propriedades de cada sistema para organizar elementos.
